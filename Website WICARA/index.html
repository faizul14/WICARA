Background About WICARA Application

We are planning to make a regional language translator application that can translate regional languages ​​into Indonesian in speech to text (and vice versa). The goal is to overcome the extinction of regional languages ​​and as a communication link between local residents and migrants in Indonesia. We provide regional languages ​​(such as Javanese, Sundanese, and Balinese) but there's a possibility to add regional languages ​​and other features . For the future we will add a feature to scan images in which there is text to translate regional languages ​​if possible.
